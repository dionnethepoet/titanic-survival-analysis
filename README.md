# titanic-survival-analysis
Exploratory data analysis of Titanic passenger survival using Python, pandas, seaborn, and matplotlib.
# Titanic Survival Analysis

## Overview
This project explores factors that influenced passenger survival on the Titanic using Python.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## Key Insights
- Females had significantly higher survival rates than males (~74% vs ~19%)
- First-class passengers were more likely to survive than lower classes
- First-class women had the highest survival rate (~97%)
- Age had a weaker impact compared to gender and class

## Visualizations
The project includes visualizations showing:
- Survival rate by gender
- Survival rate by passenger class
- Age distribution by survival

## Conclusion
Gender and passenger class were the strongest predictors of survival, suggesting evacuation priority may have been influenced by these factors.
